# Colors

Seven semantic tokens. Use these in components — never the raw palette. Each token resolves to a Tailwind utility (`bg-canvas`, `text-muted`, …) and a CSS variable (`var(--color-canvas)`, `var(--color-muted)`, …). Light and dark themes swap automatically via `data-theme="dark"` on `<html>` — toggle the theme from the Storybook toolbar to preview.

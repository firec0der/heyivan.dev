# Motion

Three duration tokens and two easing curves cover every transition on the site. The `--motion-*` namespace is not a recognized Tailwind theme key, so no utility families are generated automatically — reference these tokens directly via `var()` in CSS or inline `style` props. A global `prefers-reduced-motion: reduce` rule collapses every transition to 1ms regardless of the value chosen.

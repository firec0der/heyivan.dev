# Spacing

Twelve tokens form the rhythm scale used across components and long-form prose. Reach for the named tokens (`p-sm`, `gap-2xs`, `mb-lg`) instead of arbitrary pixel values — they keep type rhythm consistent across the site. Each token resolves to a Tailwind utility family (`p-*`, `m-*`, `gap-*`, `px-*`, `py-*`, …) and a CSS variable (`var(--spacing-sm)`, …).
